## 📊 Interactive Retail Sales Analytics Dashboard

### About
Interactive dashboard analyzing 1000+ retail sales records 
to identify trends and patterns using Python and Streamlit.

### Tools Used
- Python, Pandas, NumPy
- Streamlit, Plotly
- Microsoft Excel

### Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Interactive visualizations
- Sales trend identification

### Dataset
Retail sales dataset (1000+ records)
